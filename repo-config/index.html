Switchboot Package Repository

APT Repository (Debian/Ubuntu)

To add this repository to your system:

# Add GPG key
wget -qO- https://moooozi.github.io/switchboot/deb/switchboot-archive-keyring.gpg | sudo apt-key add -

# Add repository
echo "deb https://moooozi.github.io/switchboot/deb stable main" | sudo tee /etc/apt/sources.list.d/switchboot.list

# Update and install
sudo apt update
sudo apt install switchboot

RPM Repository (Fedora/RHEL/CentOS)

To add this repository to your system:

sudo wget -O /etc/yum.repos.d/switchboot.repo https://moooozi.github.io/switchboot/rpm/switchboot.repo

Then install Switchboot:

sudo dnf install switchboot

Or for older systems:

sudo yum install switchboot

Manual Installation

You can also download packages directly from the GitHub releases page.
